# Election Data Analysis & Visualization – 2019

## Overview
This project analyzes the Indian General Election 2019 data using Statistical Analysis, Tableau Visualization, and SQL (BigQuery).

## Objectives
- Analyze voting patterns
- Compare party performance
- Study winner vs runner-up margins
- Understand electoral competition across states

## States Analyzed
- Gujarat
- Karnataka
- Maharashtra
- Rajasthan
- Tamil Nadu

## Tools & Technologies
- Tableau
- SQL (BigQuery)
- Statistical Analysis
- Data Visualization

## Project Features
- Party-wise Vote Percentage Analysis
- Winner vs Runner-Up Comparison
- State-wise Election Overview
- Tableau Dashboards
- SQL Queries using BigQuery
